Knowledge
  1. Big O notation is a convention used to compare the efficency of algorithms. It represents the rate at which the execution time increases relative to the input size.
  2. Binary search works by checking the middle item in an array and seeing of the search for item is above or below that point. It then recursively searches either the upper or lower halves of the array until the middle item is equal to the searched item or the array size is zero.
  3. A linear search is better when considering best case.
  4. A binary search is better when considering worst case.
  5. A linear search is O(n), while a binary search is O(logn).
Thinking
    1. Binary: (log2(10000) - 1) × 6 μs = 74 μs
      Linear: (10000 / 2) × 5 μs = 25 ms
    2. Binary: (log2(10) - 1) × 6 μs = 14 μs
      Linear: (10 / 2) × 5 μs = 25 μs
  2. Always makes sense to use a binary search in this case.
